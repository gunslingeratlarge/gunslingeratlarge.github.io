NLTK

2018-05-28
机器学习
nlp, 机器学习

介绍

conda install nltk

  1. tokenize：分词
  2. stopwords：停用词
  3. stem：取词根
  4. part of speech tagging：词性标注
  5. chunking：分块
  6. chinking： 分块的对立面，从chunking中去除某些部分
  7. Name Entity Recognition：识别名词（词组）
    本篇笔记基于https://pythonprogramming.net/tokenizing-words-sentences-nltk-tutorial/ 完成。
