建立私链并部署合约

2018-04-08
区块链
以太坊, 区块链
  1. 安装truffle
    npm install -g truffle
  2. 安装测试链
    npm install -g ganache-cli
  3. 初始化truffle
    truffle init
  4. 将智能合约加到contracts中，并将deploy的js脚本加到migrations中
  5. 配置truffle
    注意windows下要删去truffle.js。将后面的配置信息拷贝到truffle-config.js
  6. 编译合约
    truffle compile
  7. 部署合约
    truffle migrate —network development
    —network 后面就跟上网络名（配置在truffle-config.js中）
  8. 下载geth
    注意版本差异
  9. 初始化创世区块
    geth —datadir .\chaindata init .\genesis.json
  10. 开启私链
    geth —datadir .\chaindata\ —rpc
  11. 新建账户
    eth.accounts
    personal.newAccount
  12. 开始挖矿
    miner.start()
  13. truffle migrate —network ourTestNet
    这次部署到geth的私链上

geth —datadir ./chaindata —rpc
geth attach \.\pipe\geth.ipc
